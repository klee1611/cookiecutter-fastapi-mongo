This is a backend boilerplate designed for high-performance RESTful APIs using FastAPI and Motor, an asynchronous driver for MongoDB. It includes health checks, CRUD operations, automatic API documentation, Docker support, and uses Cookiecutter for easy project generation.

üöÄ Quick Start

Create a new backend service in seconds with Cookiecutter:

cookiecutter gh:klee1611/cookiecutter-fastapi-mongo

After project creation, all dependencies are automatically installed!

üìã Prerequisites

  • Python 3.10+

  • Docker installed

üéì Usage Guide

Run Tests

make test

Run Development Server

make dev

This will start a MongoDB container and the FastAPI development server.

Build Docker Image

make docker-build

Docker Compose

# Start services
make docker-compose-up

# Stop services
make docker-compose-down

Access API Documentation

After starting the server, visit http://localhost:8888/docs
